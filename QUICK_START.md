# מדריך התחלה מהירה

## דקה אחת להתחלה

### 1. קבל מפתח API של Gemini
```bash
# עבור לכתובת זו וקבל מפתח חינם:
https://makersuite.google.com/app/apikey
```

### 2. הפעל את האפליקציה
```bash
npm install
npm run dev
```

### 3. פתח בדפדפן
```
http://localhost:5173
```

### 4. הכנס את מפתח ה-API
- לחץ על כפתור "API Key" בראש הדף
- הדבק את המפתח שקיבלת

### 5. צור סיכום ראשון
1. העלה קובץ וידאו (MP4, AVI, MOV, MKV)
2. הזן כותרת וז'אנר
3. כתוב תיאור קצר
4. לחץ "צור סיכום חכם"

## תכונות מתקדמות

### למידה מיוטיוב
```
1. הפעל את הסגלגל "למידה מיוטיוב"
2. הזן מזהה ערוץ YouTube (למשל: UCxxxxxxxx)
3. המערכת תלמד מהסגנון של הערוץ
```

### חיפוש אינטרנט אוטומטי
```
✓ הפעל את "חיפוש באינטרנט"
✓ המערכת תמצא מידע על הסרט אוטומטית
✓ הסיכום יהיה מדויק יותר
```

### המלצות AI
```
בחר ז'אנר → קבל המלצות על:
- אורך קטעים אופטימלי
- מרווחי חיתוך מומלצים
- סגנון עריכה מתאים
```

## טיפים מהירים

### לסיכום איכותי
```
✓ כתוב תיאור מפורט (3-5 משפטים)
✓ ציין רגעים חשובים בסרט
✓ הזכר דמויות ראשיות
✓ תאר את הז'אנר והאווירה
```

### להאצת עיבוד
```
✓ השתמש בקבצים דחוסים
✓ הורד איכות וידאו אם אפשר
✓ סגור תוכניות כבדות אחרות
```

### לשיפור מתמיד
```
✓ דרג כל סיכום שיוצר
✓ השאר משוב
✓ המערכת לומדת מהדירוגים
```

## בעיות נפוצות

### "שגיאה בעיבוד הווידאו"
```
פתרון:
1. ודא שהקובץ תקין
2. נסה פורמט אחר
3. הקטן את גודל הקובץ
```

### "מפתח API לא תקין"
```
פתרון:
1. בדוק שהעתקת את כל המפתח
2. ודא שהמפתח פעיל ב-Google AI Studio
3. נסה ליצור מפתח חדש
```

### "הדפדפן איטי"
```
פתרון:
1. סגור טאבים אחרים
2. השתמש ב-Chrome או Firefox
3. ודא שיש לפחות 4GB RAM פנויים
```

## קיצורי מקלדת

```

Escape          : בטל פעולה נוכחית
```
